Program służy do dzielenia dużych plików .pdf na mniejsze pliki pdf - określone przez użytkownika.

Wymagania:
- python 3.x 

Założenia:
- pliki nazwy.txt, Dzielenie_etykiet.pyc i docelowo dzielony plik pdf powinny być w jednym folderze
- podaj zmienne
- po poprawnym przejściu programu, pliki wyjściowe (podzielone pdfy) powinny znajdować się w katalogu /output
